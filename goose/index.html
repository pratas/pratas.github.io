Introduction

What is GOOSE ?

GOOSE is a framework for DNA data manipulation and analysis.

It answers the following questions:
  • Can GOOSE manipulate any genomic sequence? (yes)
  • Can GOOSE filter, simulate, transform, compact, shuffle genomic sequences? (yes)
  • Can GOOSE process SEQ (ACGTN), FASTA, Multi-FASTA and FASTQ files? (yes)
  • Can GOOSE support pipes for easy integration with current pipelines? (yes)

How do I get GOOSE ?

Clone our repository and run make command: 
git clone https://github.com/pratas/goose.git
cd goose/src/
make
